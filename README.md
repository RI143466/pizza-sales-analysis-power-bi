# 🍕 Pizza Sales Analysis Dashboard
## Project Overview
This project analyzes pizza sales data to identify:
- Revenue trends
- Customer ordering behavior
- Top-selling pizza categories
- Peak sales periods
The dashboard was built using Power BI and Python to generate business insights from sales data.
---
## Tools Used
- Python
- Power BI
- Excel
- Jupyter Notebook
---
## Dashboard Features
- Total Revenue KPI
- Total Orders Analysis
- Sales Trend Visualization
- Category-wise Performance
- Top-selling Pizzas
- Daily and Monthly Sales Insights
---
## Key Insights
- Weekends recorded highest sales
- Large pizzas generated maximum revenue
- Top pizza categories contributed most profits
- Peak ordering hours improved business understanding
